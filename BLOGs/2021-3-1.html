ユーグリッドの互除法をPythonでやってみよう

科学部が教えるわかりやすいプログラミング講座

まずはユーグリッドの互除法のアルゴリズムを考えてみます。
～～～
Pythonに限らずループといえばfor文ですが、Pythonでのfor文は以下のように使われて柔軟な条件設定ができません。

        
for w in "Hello":
    print(w)

とこのようにPythonでfor文はリストからデータを出すときに使います。
しかしwhile文では次のように設定できます。
        
while 条件:
    条件が正しいときに実行する文
else:
    条件が正しくないときに実行する文
これをユーグリッドの互除法に当てはめると
        
while a % b != 0:
    c = a
    b = a % b
    a = c
となります。
ここで
!=は左辺と右辺が等しくない、
a % bはaをbで割った余りを表します。
また
a = b
b = a % b
ではなく 
c = a
b = a % b
a = c
としているのはa = b としてしまうとa % b のaもbになり常に1となるためです。

おまけの話

whileだけでも十分ですがこれだけだととても使いにくいです。
まずはじめに扱いやすいように関数として実装しましょう。
        
def gcd(a,b):
    while a % b != 0:
        c = a
        b = a % b
        a = c
    return b
次に関数の引数の大きさが逆でも正常に動作するようにします。
        
def gcd(a,b):
#aとbの大きさを揃える
    if a > b:
        c = b
        d = a
    else:
        c = a
        d = b
    a = c
    b = d
#while文でのループ
while a % b != 0:
    c = a
    b = a % b
    a = c
return b
後は入力ウィンドウと最大公約数を表示出来るようにします。
input()はintではない点にも注意しましょう。
以下がPythonでのコードです。
        
#最大公約数を求める関数
def gcd(a,b):
#aとbの大きさを揃える
    if a > b:
        c = b
        d = a
    else:
        c = a
        d = b
    a = c
    b = d
#while文でのループ
    while a % b != 0:
        c = a
        b = a % b
        a = c
    return b

#入力ウィンドウ
a = input("数字を入力してください>>>")
b = input("数字を入力してください>>>")

#字体を整える
a = int(a)
b = int(b)

#最大公約数を表示
print(gcd(a,b))

#ウィンドウを閉じさせない
input("何かボタンを押してください")
